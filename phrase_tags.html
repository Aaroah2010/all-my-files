Phrase Tags

Phrase tags are used to write some piece of text that has some certain meaning . In other words, they are used to indicate structural meaning of a text.

1. Abbreviation tag - abbr tag

HTML

2. quote tag - q tag, used to write in double quotes

Hello there

3. text direction tag - bdo tag ( bi-directional override )

My name is Keshav .

4. blockquote tag - used to write quoted sentences

Imagination is more important than knowledge.

5. Address tag - to write the address of someplace

I live in
New Delhi, India

6. definition tag - dfn tag, to write definition of a keyword

CSS is used for designing the webpage.

7. cite tag - used to write title of something creative like a story, book, porem etc

Harry Potter by JK Rowling
